# Contributing to Rick-And-Morty-Loop

We welcome contributions from the multiverse (except from Council of Ricks members). 

## How to Contribute
1. **Fork the Repository**: Create your own workspace.
2. **Create a Feature Branch**: `git checkout -b feature/amazing-new-logic`.
3. **Submit a PR**: Ensure your code passes the internal Rick-Oversight audit.

## Standards
- All code must be adversarial-tested.
- No mocks. No placeholders.
- If it breaks the build, you fix the build.

**Shut up and compute.**
